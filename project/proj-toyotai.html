ToyotAI

Overview

ToyotAI makes choosing a Toyota model engaging and efficient: a playful personality quiz maps lifestyle traits to suitable models; an AI virtual salesman answers nuanced questions; and a traditional filtering interface supports power users needing granular control.

Highlights

  • Quiz-to-model mapping engine with weighted trait scoring
  • LangChain-powered conversational agent for spec + lifestyle Q&A
  • Hybrid retrieval: structured filtering + semantic embedding search
  • Data pipeline using Pandas to normalize model specs & features
  • Hackathon delivery under 24 hours with iterative user testing

Tech Stack

Python • LangChain • Pandas • React • CSS • Vector Search

Architecture Notes

Backend services expose quiz inference and chat endpoints. Vehicle dataset standardized (feature flags, numeric normalization) pre-indexed for hybrid retrieval. Conversation agent chains spec lookup + recommendation rationale. Frontend React components provide multi-path (quiz / chat / filters) discovery funnel improving retention and satisfaction.
