Overview

Developed at TidalHack 2025, this project addresses the critical time lost between wildfire detection and coordinated response. It uses a network of AI agents that autonomously detect anomalies, gather contextual data, learn from past wildfire events, and generate an actionable response plan‚Äîall within roughly 120 seconds.

How It Works

  • Detection Agent: Analyzes satellite imagery to spot potential wildfire anomalies.
  • Enrichment Agent: Gathers contextual data including weather, humidity, wind direction, local population, fire stations, and evacuation zones.
  • Learning Agent: Compares new anomalies against historical wildfire events and refines decision-making with each iteration.
  • Action Agent: Generates a recommended course of action and disseminates it via AWS SNS to fire stations and disaster relief centers.

Technical

Built using AWS Bedrock, AgentCore, AWS Titan, Google Gemini, and integrated with DynamoDB, S3, and SNS for orchestration, persistence, and communication. The agents collaborate asynchronously to build a context-aware, continuously learning decision engine.
