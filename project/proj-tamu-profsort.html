Tamu-ProfSort

Overview

Electron-based application leveraging headless scraping and caching to present comparative grading metrics across professors and courses.

Project Link: bit.ly/tamuprofsort ↗

Highlights

  • Automated Puppeteer scraping with delta-detection to minimize requests
  • Local persistent cache & periodic refresh scheduling
  • Interactive sorting & filtering by distribution metrics

Stack

Electron • JavaScript • Puppeteer • Charting
